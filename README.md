# ARcricInfo

**ARcricInfo** is a modern, responsive web application built to provide cricket fans with up-to-date match information, player stats and team coverage — inspired by platforms like Cricbuzz.  
This project is developed as part of my final-year Computer Science work.

## Live Demo  
Try it out at: https://arcricinfo1.netlify.app/ 

## Features  
- Real-time or near real-time scoreboard for live matches  
- Player profiles and team statistics  
- Interactive and responsive UI designed for desktop & mobile  
- Built with a focus on Gen Z audience: clean layout, easy navigation, modern design  
- Admin panel (planned / work-in-progress) for adding match data, player updates etc (if applicable)  

## Tech Stack  
- **Frontend**: HTML, Tailwind CSS, JavaScript  
- **Backend**: Node.js (Express) + MySQL  
- **Deployment**: Netlify (for frontend demo)  
- **Version Control**: Git & GitHub
